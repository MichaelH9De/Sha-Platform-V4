# Roadmap

## Deployment baseline (complete)

PostgreSQL migrations, Entra OIDC, project memberships, transactional audit, protected routes, live project/dashboard reads, validated workflows, tests and deployment controls.

## Commercial Release

Fee budgets, fee forecasts, variation workflow, invoicing milestones, margin-risk dashboards and resource forecasting.

## QA/BIM Release

QA review workflows, deliverable issue gating, BIM model records, CDE references and model QA checks.

## Reporting Release

Internal dashboards, client-facing reports, field allowlists, scheduled reports and report approval workflow.

## Integration Release

Microsoft 365, SharePoint/CDE, Teams, Outlook, Power BI, accounting, time-recording and BIM/Revit metadata integrations.

## Intelligence Release

AI summaries, risk detection, lessons learned search and meeting action drafting with source citations and human approval.
